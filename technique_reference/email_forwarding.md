# Email Forwarding: don@mindless-scribbles.com → Gmail

Domain registered with Squarespace. No paid service needed — forwarding is built in.

## Setup forwarding (Squarespace)

1. Log into squarespace.com
2. **Domains** → click `mindless-scribbles.com`
3. **Email** → **Add Forwarding Address**
4. Set: `don` → `your@gmail.com`
5. Save — active within a few minutes

## Send as don@mindless-scribbles.com from Gmail (optional)

1. Gmail → **Settings (gear)** → **See all settings**
2. **Accounts and Import** → **Send mail as** → **Add another email address**
3. Enter `don@mindless-scribbles.com`
4. Gmail sends verification email → it forwards to your Gmail → click the link
5. Done — can now reply as `don@mindless-scribbles.com`
