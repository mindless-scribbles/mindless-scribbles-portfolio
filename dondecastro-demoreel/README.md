# dondecastro-demoreel — Reference Files

This folder contains the design decisions and scaffold from a prior planning session for **dondecastro.com**.

When starting the `dondecastro-demoreel` Astro repo, use these files as the starting point:

- **CLAUDE.md** — project overview, site structure, design direction, content schemas, and conventions for Claude Code
- **decisions.md** — architecture decisions with reasoning (Astro over Next.js, video embedding strategy, typography choices, etc.)
- **QUICKSTART.md** — step-by-step guide for initializing the Astro project and working with Claude Code
- **scaffold.sh** — shell script that creates the full directory structure, content collection config, base layout, VideoEmbed component, TagFilter component, and global CSS

## To use in a new session

1. Create a new GitHub repo named `dondecastro-demoreel`
2. Copy these files into it
3. Follow the steps in `QUICKSTART.md`
