# mindless-scribbles.com

Personal playground for experiments, blog posts, and interactive demos. Built with Next.js, Tailwind CSS v4, and the Netlify/Stackbit visual editor.

## Getting Started

```bash
npm install
npm run dev
```

Dev server runs at `http://localhost:3000`.

## Deployment

Intended for deployment on Vercel. Connect the GitHub repo in the Vercel dashboard — it auto-detects Next.js with no configuration needed.
